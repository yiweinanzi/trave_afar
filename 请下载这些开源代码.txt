========================================
GoAfar项目 - 需要下载的开源代码清单
========================================

项目已完成基础功能，现在需要LLM4Rec相关代码来增强推荐效果。

========================================
方案 A: 轻量级LLM增强（推荐先做）
========================================

只需下载3个项目：

1. Qwen2.5 (必需) ⭐⭐⭐
   链接: https://github.com/QwenLM/Qwen2.5
   用途: 轻量级中文大模型，用于意图理解和文案生成
   命令: git clone https://github.com/QwenLM/Qwen2.5.git

2. LangChain (推荐) ⭐⭐
   链接: https://github.com/langchain-ai/langchain
   用途: LLM应用开发框架，简化LLM集成
   命令: git clone https://github.com/langchain-ai/langchain.git

3. LLM4Rec (推荐) ⭐⭐⭐
   链接: https://github.com/WLiK/LLM4Rec
   用途: 学习LLM在推荐系统中的最佳实践
   命令: git clone https://github.com/WLiK/LLM4Rec.git

下载完成后放到: /root/autodl-tmp/goafar_project/open_resource/

========================================
方案 B: 深度LLM4Rec集成（进阶）
========================================

在方案A基础上再下载：

4. TALLRec ⭐⭐
   链接: https://github.com/SAI990323/TALLRec
   用途: POI属性理解和结构化表征
   命令: git clone https://github.com/SAI990323/TALLRec.git

5. RecLLM ⭐⭐
   链接: https://github.com/HKUDS/RecLLM
   用途: 对话式推荐和推荐解释生成
   命令: git clone https://github.com/HKUDS/RecLLM.git

6. ChatRec ⭐
   链接: https://github.com/RUCAIBox/ChatRec
   用途: 对话式推荐系统参考
   命令: git clone https://github.com/RUCAIBox/ChatRec.git

========================================
方案 C: 多模态增强（高级）
========================================

7. LLaVA (多模态)
   链接: https://github.com/haotian-liu/LLaVA
   用途: 处理景点图片，图文推荐
   命令: git clone https://github.com/haotian-liu/LLaVA.git

8. LlamaIndex (RAG)
   链接: https://github.com/run-llama/llama_index
   用途: 文档索引和知识库
   命令: git clone https://github.com/run-llama/llama_index.git

========================================
推荐下载顺序
========================================

建议按以下顺序下载：

第一批（立即下载）:
  1. Qwen2.5
  2. LLM4Rec  
  3. LangChain

第二批（之后下载）:
  4. TALLRec
  5. RecLLM

第三批（可选）:
  6. LLaVA
  7. LlamaIndex

========================================
模型下载（Qwen2.5-0.5B）
========================================

方法1: 使用ModelScope（推荐，国内快）
python -c "
from modelscope import snapshot_download
model_dir = snapshot_download(
    'qwen/Qwen2.5-0.5B-Instruct',
    cache_dir='/root/autodl-tmp/goafar_project/models'
)
print(f'模型下载到: {model_dir}')
"

方法2: 使用HuggingFace
from transformers import AutoModelForCausalLM, AutoTokenizer
model = AutoModelForCausalLM.from_pretrained(
    "Qwen/Qwen2.5-0.5B-Instruct",
    cache_dir="/root/autodl-tmp/goafar_project/models"
)

========================================
下载后要做的事
========================================

1. 学习参考代码
   - 查看 LLM4Rec/ 的实现
   - 查看 Qwen2.5/examples/ 的用法

2. 完善我们的代码
   - 实现 src/llm4rec/ 的LLM调用逻辑
   - 集成Qwen到内容生成模块

3. 测试运行
   - 测试意图理解
   - 测试LLM Reranking
   - 测试文案生成

4. 对比效果
   - 模板模式 vs LLM模式
   - 准确率和用户体验

========================================
当前项目状态
========================================

✅ 已完成:
  - 完整的模块化架构
  - BGE-M3语义检索（可运行）
  - OR-Tools路线规划（可运行）
  - 模板文案生成（可运行）
  - LLM4Rec框架（待填充实现）

⏳ 等待:
  - 下载LLM4Rec相关开源代码
  - 学习并实现LLM调用逻辑
  - 下载Qwen模型

✨ 可以先运行基础版:
  bash quick_start.sh（不需要LLM）
