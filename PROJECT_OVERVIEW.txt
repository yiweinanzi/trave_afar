💡 功能对比
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  当前版本（模板模式）          vs          LLM4Rec增强版
  ────────────────────────────────────────────────────────────
  
  意图理解:
    关键词匹配 (~60%准确率)        →    LLM深度理解 (~85%准确率)
  
  召回策略:
    固定权重合并                   →    基于意图动态调整
  
  重排序:
    简单规则                       →    LLM理解协同性
  
  文案生成:
    模板填充                       →    LLM个性化生成
  
  可解释性:
    弱                            →    强（详细推荐理由）

🚀 快速开始

  当前可以运行基础版:
  
    cd /root/autodl-tmp/goafar_project
    source /root/miniconda3/bin/activate goafar
    bash quick_start.sh
  
  或分步运行:
  
    # 测试意图理解（模板模式）
    python src/llm4rec/intent_understanding.py
    
    # 测试LLM Reranker（规则模式）
    python src/llm4rec/llm_reranker.py
    
    # 测试向量构建
    python src/embedding/vector_builder.py
